# 柏基投资（Baillie Gifford / Scottish Mortgage）投资框架

来源：Scottish Mortgage 2025年报（截至2025年3月31日）

## 核心哲学：寻找异常值（Outliers）

> "大多数公司根本不重要。研究表明，只有少数公司使得投资股票市场变得有价值。"
> — Lawrence Burns, Deputy Manager

## 异常值公司的四大特征

### 1. 非常规性（Unconventional）
- 通过普通方法不可能实现非凡成果
- 案例：SpaceX让火箭可重复使用，以竞争对手时间和成本的一小部分开发火箭
- "支持非常规公司的意愿是投资异常值的必要条件"

### 2. 长期聚焦（Long-Term Focused）
- Amazon多年选择不盈利而是不断再投资
- Hermes家族承诺至少到2041年不出售任何股份
- 柏基前10大上市持仓中8家持有超过5年

### 3. 大机遇（Large Opportunities）
- 寻找"多幕剧"公司：
  - MercadoLibre：电商→金融科技（占收入40%+）→可能的广告平台
  - Spotify：音乐→播客→有声书→教育
  - Amazon：在线书商→云服务→AI基础设施
- Wise只服务了不到5%的消费者跨境市场和不到1%的企业跨境市场

### 4. 适应性（Adaptable）
- 美团从团购网站的5000家公司"克隆战争"中胜出
- Hermes用一个世纪从马具制造商转型为奢侈手袋品牌
- Meta和Spotify在后疫情时代快速重组，专注效率和盈利

## 买入决策框架

两个核心问题：
1. **公司所面对的机会是否具有变革性，且正在变大还是变小？**
2. **捕获该机会的可能性是在增加还是减少？**

买入信号：
- 机会正在变大+捕获概率在增加 → 买入/加仓
- 非常规的方法+巨大的TAM+适应性强的团队
- 估值不是首要考虑——关键是5-10年后的终局

## 卖出纪律

卖出NVIDIA的逻辑（从最大持仓减至2.3%）：
- AI要真正变革性就必须变得普遍 → 商品化
- "一个建立在7万美元芯片和60%利润率上的世界可能不可持续"
- 追求**不对称结果**：当风险/回报变得均衡时卖出
- 不是对公司的轻视，是对不对称性的纪律

卖出触发条件：
- 机会在缩小
- 捕获概率在降低
- 公司变得"常规化"
- 风险/回报从不对称变为均衡
- 发现更好的不对称机会

## 不对称思维（核心中的核心）

柏基过去10年：
- 前10名贡献者回报：6倍到80倍
- 后10名贡献者：基本都是-96%到-100%
- **接受损失是策略的一部分——关键是确保赢家的规模远大于输家**

## 对韧性的强调

> "在一个日益碎片化、更加保护主义、更不可预测的世界中，组织灵活性将比以往任何时候都重要。"

最有希望的企业共享的特质不仅是速度或规模，而是"吸收冲击、快速学习和在不失去动力的情况下重新定向的能力"。

## 私人公司投资（27.7%配置）

- 自2012年以来部署46亿英镑
- 51家当前持有，39家已上市，8家被收购，1家清算
- 前5家占私人公司敞口的61%
- 提供"通往未来的窗户"——获取差异化的洞见
- 估值：每季度重估，独立于投资经理，参考S&P Global第三方意见

## 杠杆策略

- 净杠杆率：13%
- 加权平均债务成本：3.1%
- 不试图通过频繁调整杠杆来把握短期市场时机
- 债务结构多元化：短期循环信贷+长期固定利率+超长期私募票据（到2062年）

## 费用极低

- Ongoing charges ratio：0.31%（行业最低水平之一）
- "限制费用是对股东的首要义务"

## 失败案例的坦诚反思

Northvolt（全额减记，-100%）：
- "尽管欧洲对电池制造有深层需求，公司未能扩大生产规模"
- "关于资本密集型事业和制造业中执行重要性的深刻教训"

## 当前重点投资主题

1. **AI从理论到实践**：计算需求结构性增长，但芯片可能商品化
2. **中国复兴**：估值从"定价为毁灭"开始恢复，约14%配置
3. **新兴市场数字金融**：拉美和东南亚渗透率仍极低
4. **下一代交通**：自动驾驶卡车、飞行出租车、无人机物流
5. **物理世界创新**：SpaceX证明"物理世界仍为创新提供巨大空间"
