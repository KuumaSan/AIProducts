# 《证券分析》核心决策框架（格雷厄姆/多德）

## 三条铁律

1. **投资 = 深入分析 + 本金安全 + 满意回报**（不符合就是投机）
2. **安全边际是一切的基础**：买入价必须远低于保守估计的内在价值
3. **市场短期是投票机，长期是天平**：价格偏离价值是常态，也是机会来源

---

## 固定价值投资决策树（债券/优先股）

**第一关：企业偿债能力（不看抵押权，看盈利）**
- 盈利保障倍数（7年均值，总扣除法）：公用事业≥1.75倍，铁路≥2倍，工业≥3倍
- 股票市值/债券债务：公用事业≥50%，铁路≥67%，工业≥100%
- 不合格→拒绝

**第二关：萧条考验**
- 在不利经济条件下仍能通过第一关？否→拒绝

**第三关：不以收益换安全**
- 高票面利率弥补安全性不足？是→拒绝

**第四关：具体标准**
- 规模：公用事业收入≥200万，工业销售≥500万
- 营运资本（工业）：流动比率≥2，净流动资产≥债券
- 折旧充足性：公用事业≥12%毛利润，铁路维修≥32%
- 银行借款/即将到期债务→警示

**第五关：选择最优**
- 同一稳健公司→买收益率最高的（通常是次级债券）
- 收益差不大→选优先级更高的
- 优先股→必须满足债券标准+额外安全边际
  - 优先股保障倍数：公用事业≥2倍，铁路≥2.5倍，工业≥4倍

## 普通股投资决策树

**第一关：盈利能力验证**
- 5-10年平均盈利（剔除非经常项目、调整折旧）
- 用资产负债表交叉验证（盈余变化vs报告收益）
- 用联邦所得税检验法验证
- 不可靠→放弃

**第二关：定量筛选**
- 市盈率≤20倍（平均盈利基础），前景一般→12-12.5倍合适
- 收益稳定（10年内波动可控）
- 财务安排审慎（营运资金比率≥2:1）

**第三关：资产负债表检验**
- 价格vs账面价值→极端溢价需警惕
- 价格vs流动资产价值→低于则可能严重低估
- 价格vs现金资产价值→低于则极端低估
- 清算价值估算：现金100%+应收75-90%+存货50-75%+固定资产15%

**第四关：定性因素**
- 企业内在稳定性（行业特征，非仅统计数据）
- 管理层诚信和能力
- 竞争地位和护城河
- 会计政策可疑→远离所有证券

**第五关：价格决策**
- 私人企业价值试金石：价格≤企业家愿付价格+20%流动性溢价
- 低于净流动资产→特别机会（确认资产未快速流失）
- 收入来源分析：固定收入用高乘数+经营收入用低乘数

**第六关：卖出决策**
- 可转换证券溢价超25-35%→卖出
- 股价远超正常值（如正常值的4/3以上）→考虑卖出
- 投机性资本结构公司股价超100美元→不再具有优势

## 损益表分析要点

- **报告利润≠可投资价值**：真正要资本化的是"质量调整后的可持续经营利润"
- 剔除：一次性项目、会计政策变化、费用资本化、异常收入确认
- **折旧验证**：用纳税申报数据或同行比较；公用事业折旧<10%毛利润可能不足
- **子公司盈利**：坚持用合并报表；母公司单独报表常严重误导
- **趋势陷阱**：趋势是事实，"未来趋势"只是假设，不可无限延伸
- **稳定性比趋势更重要**：看行业特征决定的内在稳定性，不仅看统计数字

## 资产负债表分析要点

- **三层估值**：账面价值→流动资产价值→现金资产价值
- 账面价值只是起点；真正的下限看保守折减后的可变现资产
- **营运资本三标准（工业）**：现金充裕+流动比率≥2+营运资本≥债券债务
- 银行借款是隐藏风险信号（萧条时银行收贷加剧困境）

## 特殊机会识别

| 机会类型 | 识别信号 |
|----------|---------|
| 低于清算价值 | 市价<净流动资产，且有合理平均盈利 |
| 收入来源错配 | 大部分收入来自投资/租金，用更高乘数估值 |
| 接管/重组 | 高级证券极低价格，分析重组后地位 |
| 可转换套利 | 可转换证券以投资级价格且接近转换平价→必然优于普通股 |
| 诉讼压制 | 下跌与诉讼实际影响不成比例 |
| 资本结构投机 | 普通股占比极小+股价超跌→上涨空间远大于下跌 |
| 同行业低估 | 比较分析显示显著统计优势+定性验证 |

## 风险红旗

| 信号 | 应对 |
|------|------|
| 可疑会计政策 | 远离该公司所有证券 |
| 管理层与股东利益冲突 | 审视薪酬、扩张、分红 |
| 高杠杆 | 评估萧条情景偿债能力 |
| 大量银行借款/到期债务 | 无明确再融资能力→回避 |
| 新时代理论/趋势外推 | 拒绝为增长支付无限溢价 |
| 市盈率>20倍 | 属投机非投资 |
| 报告利润与资产负债表矛盾 | 用资产负债表检验损益表 |
| 折旧不足 | 用税务数据或同行比较验证 |
